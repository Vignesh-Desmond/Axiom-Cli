# axiom-cli
Proctoring application based on YoloV3, Darknet, VGGFACE, MTCNN

Run `pip install -r requirements.txt` to install all required packages and dependecies

[final.py](https://github.com/Vignesh-Desmond/axiom-cli/blob/main/final.py) is the combined src file.
Before running, uncomment weights_download() to download yolov3.weights (first time only)
